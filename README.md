### frontend-project 개발환경

1. 프로젝트 환경설정(vite를 활용한 React 설치): `npm create vite@latest (React TypeScript + React Compiler)` <br />

2. React 중앙집중식 상태관리 라이브러리 Zustand 설치: `npm install zustand` <br />

3. 외부 오픈 API 통신을 위한 라이브러리 Axios 설치: `npm install axios` <br />

4. React Router 설치: `npm install react-router-dom localforage match-sorter sort-by` <br />

5. React Toast Popup 모듈 설치 : `npm install react-simple-toasts` <br />

6. 그리드 Resize 모듈 설치 : `npm install react-resize-layout react react-dom` <br />

7. 다국어 모듈 설치 : `npm install i18next react-i18next i18next-browser-languagedetector` <br />

8. AG-Grid 모듈 설치 : `npm install ag-grid-community ag-grid-react` <br />

   - 집계 모듈 설치: `npm install ag-grid-enterprise@34.3.1` <br />

9. Ant Design 모듈 설치 : `npm install antd` <br />

   - React 19 적용 패치 : `npm install @ant-design/v5-patch-for-react-19`<br />

10. CSS 스타일링을 위한 SASS/SCSS 설치: `npm install -D sass` <br />

- className을 조건부 모듈 설치 : `npm install --save clsx` <br />
- Styled Components 모듈 설치 : `npm i styled-components` <br />

11. js-cookie 모듈 설치 : `npm install js-cookie @types/js-cookie`

설치
npm i
npm install

개발 서버 실행
npm run dev

프로덕션용 빌드
npm run build

빌드 결과 미리보기기
npm run preview
